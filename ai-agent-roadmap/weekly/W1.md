## Day 3 · Thu 5.14 · 2h · ReAct Loop v1（核心）

### 资料（~30 min）

1. [ReAct 论文 Abstract](https://arxiv.org/abs/2210.03629) — 只看 abstract + section 1，10 分钟
2. [一篇中文 ReAct 实现博客](https://zhuanlan.zhihu.com/p/659572886) — 看作者怎么组织 while 循环
3. **不要** 看 LangChain ReAct——本周纯手写

### 实操（~1.5h）

- [ ] 写 `agent_v2.py`：
  - `class Agent`：持有 `messages: list`、`tools: dict[str, Callable]`、`max_iter=5`
  - `def run(self, user_query: str) -> str`：
    1. 循环最多 `max_iter` 次：
    2. 调 DeepSeek with messages + tools
    3. 看 `message.tool_calls`：
       - 没有 → `break`，返回 `message.content`
       - 有 → 执行每个 tool_call（调对应 Python 函数），append `role="tool"` 结果到 messages，进入下一轮
    4. 超过 max_iter → 返回兜底文案"任务复杂，超过最大轮次"
- [ ] 测 3 个问题：
  1. `(3+4)*5 等于多少`（需 2 步）
  2. `(2*3+4) - (10/2)`（需 4 步）
  3. `今天天气怎么样`（不应调工具，直接答）

### DoD

- [ ] 3 个测试问题全部能跑通
- [ ] 完整日志看到 `reason → act → observe → reason ...` 序列（用 `print` 标识每轮）
- [ ] 第 3 问不调用 calculator
- [ ] commit `feat: ReAct loop with max iterations`

### 自测

1. 为什么 tool 结果要以 `role="tool"` 加进 messages？加 `role="assistant"` 会怎样？
2. 你的 agent 在第 5 轮还没收到最终答案，应该怎么处理？写出兜底代码片段

## Day 4 · Fri 5.15 · 2h · 异常处理 + 3 层 Fallback

### 资料（~20 min）

1. [Pydantic ValidationError 文档](https://docs.pydantic.dev/latest/errors/validation_errors/) — 错误结构
2. 回顾 agent_v2.py，列出所有可能 fail 的点

### 实操（~1.5h）

加 3 层防御：

- [ ] **Layer 1 · tool_calls JSON 解析**：try/except `JSONDecodeError`，失败时打印警告 + 进入下一轮
- [ ] **Layer 2 · Pydantic 校验失败**：try/except `ValidationError`，把错误信息以 `role="tool"` 回传给 LLM 让它重试
- [ ] **Layer 3 · 工具执行抛异常**（如 calculator 除以 0）：try/except `Exception`，把异常字符串回传

外加：

- [ ] 加 `--debug` 参数（argparse），打开后打印每轮完整 messages
- [ ] 故意造 3 种错误测试：
  - LLM 返回非法 JSON（mock 一个坏响应）
  - Pydantic 校验失败（让 LLM 返回 `op="pow"`，可改 prompt 诱导）
  - 工具抛异常（问 `100 / 0`）

### DoD

- [ ] 3 种错误都能 fallback，不让程序崩溃
- [ ] 错误信息能回传给 LLM，让它换策略（不是直接卡死）
- [ ] commit `feat: 3-layer error fallback`

### 自测

1. 怎么从 \`\`\`json ... \`\`\` 包裹的字符串里稳健提取 JSON？写正则
2. Pydantic `ValidationError.errors()` 返回的结构是什么？怎么用它告诉 LLM "你的 op 字段不在允许范围"？
3. 死循环（LLM 反复调同一工具同一参数）怎么检测？`max_iter` 之外还需要什么机制？
